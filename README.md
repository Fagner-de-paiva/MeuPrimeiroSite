# MeuPrimeiroSite
👟 Loja de Chuteiras

Projeto desenvolvido como parte de atividade acadêmica, com o objetivo de praticar conceitos de desenvolvimento web.

📌 Sobre o projeto

Este é um site de uma loja de chuteiras, onde os usuários podem visualizar diferentes modelos e estilos de chuteiras.

O projeto foi criado com foco em aprendizado, explorando estruturação de páginas, estilização e organização de conteúdo.

🚀 Tecnologias utilizadas
HTML5
CSS3

🎯 Funcionalidades
Exibição de chuteiras disponíveis

Navegação entre páginas
